- 1.0 - STUDY -------------------------------------------------------------

	As a user I want to be able to create a new study

- 1.1 ---------------------------------------------------------------------

	As an admin of a study I want to be able to edit and remove 
	the study

- 1.2 ---------------------------------------------------------------------

	As an admin of a study I want to be able to associate and
	unassociate other users with the study. Furthermore I must
	be able to decide on their role in the study.

- 1.3 ---------------------------------------------------------------------

	As an admin of a study I want to be able to setup stages
	of the review process in the study.

- 1.4 ---------------------------------------------------------------------

	As an admin of a study I want to be able to add articles
	to the study.

- 2.0 - STAGE -------------------------------------------------------------

	As an admin of a study I want to be able to decide the
	protocol for how users input fields concerning the articles
	of the study.

- 2.1 ---------------------------------------------------------------------

	As an admin of a study I want to be able to decide whether
	users of the study function as researchers or reviewrs
	on the individual stages

- 2.2 ---------------------------------------------------------------------

	As an admin of a study I want to be able to decide
	the strategy for completing the individual stages
